3.0 Usage Notes

-  When a batch is queued, it is persisted either in the file system or in the user-implemented persistence medium. However, when the ACH file is generated, it is saved in the file system.

-  When webMethods Trading Networks is installed and the WmACHForTN package is enabled, the following notes apply:

·         The ACH Routing Engine is automatically disabled and Trading Networks Processing Rules are used.

·         The wm.ach.tn.trp:send service generates an ACH file that is first saved in the file system and then persisted in Trading Networks.

·         When the wm.ach.tn.trp:receive service is invoked, a folder named achoutput is created in the file system. The ACH file is stored in this folder for internal use apart from persisting in Trading Networks.
