Listener Notifications Screen

Use the Listener Notifications screen to view and manage the status for all listener notifications that you configured using the webMethods Developer.

This screen shows the following information:

Notification Name
Displays the name of the listener notification.
Package Name
Displays the name of the package for the listener notification.
Enabled
Indicates the availability of the listener notification specified in the Notification Name. When enabled (Yes), the listener notification is enabled and available for a listener to use. When disabled (No), the listener is not available for a listener to use.

To enable a listener, click the No link in this field.

To disable a listener, click the Yes link in this field.

 

Important! You are allowed to change the notification order of the child listeners only if a child listener is directly associated with notifications (not inheriting the notifications from the parent). The link to change the notification order is inactive for the child listeners that do not have any notifications configured.
