Listeners Screen

Use the Listeners screen to view and manage the status for all WebSphere MQ Adapter listeners currently configured on the webMethods Integration Server.

This screen shows the following information.

Listener Name
Displays the name of the listener. To display detailed information about a listener, click View. To update a listener, click Edit.
Package Name
Displays the name of the package for the listener.
Enabled
Indicates the availability of the listener specified in the Listener Name. When enabled (Yes), the listener is enabled and available for the listener notification services to use. When disabled (No), the listener is not available for the listener notification services to use.

To enable a listener, click the No link in this field.

To disable a listener, click the Yes link in this field.

 

Note: You cannot enable a child listener. If you enable a parent listener, all the child listeners associated with that parent will be enabled.

Edit
Displays the settings for the listener specified in Listener Name, and allows you to update the listenerâ€™s settings. You can only edit a listener if it is disabled.
View
Displays the settings for the listener specified in Listener Name. You cannot edit a listener from the View Listener screen.
Copy
Creates a new listener by copying the properties for an existing listener.
Delete
Deletes the listener specified in Listener Name. You can only delete a listener if it is disabled.
To create a new listener, click Configure New Listener.

 

Important! You cannot delete a child listener. If you delete a parent listener, all the child listeners associated with that parent will be deleted.
