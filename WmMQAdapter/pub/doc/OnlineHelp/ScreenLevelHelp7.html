Listener Types Screen

Use the Listener Types screen to select the listener type for the listener you are creating. The WebSphere MQ Adapter supports two listener types called WebSphere MQ Single-Queue Listener Service and WebSphere MQ Multi-Queue Listener Service.

Select WebSphere MQ Single-Queue Listener Service or WebSphere MQ Multi-Queue Listener Service to continue.

To return to the Listeners screen, click Return to WebSphere MQ Adapter Listener Types.
